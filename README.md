🌤️ Weather App

Простое веб-приложение для просмотра текущей погоды и прогноза на 5 дней по любому городу мира.

🔗 Демо: https://dex1tt.github.io/weather-app/

Возможности


🔍 Поиск погоды по названию города
🌡️ Текущая температура, ощущаемая температура, влажность и скорость ветра
📅 Прогноз погоды на 5 дней вперёд
🎨 Динамический градиентный фон, меняющийся в зависимости от погодных условий
⚠️ Обработка ошибки, если город не найден
📱 Адаптивная вёрстка под мобильные устройства


Технологии


HTML5 — семантическая структура
CSS3 — flexbox, grid, анимации, адаптивная вёрстка
JavaScript (Vanilla) — без фреймворков и библиотек
OpenWeatherMap API — данные о погоде и прогнозе


Как это работает


Пользователь вводит название города и нажимает «Найти» (или клавишу Enter)
Приложение делает запрос к OpenWeatherMap API и получает текущую погоду
Параллельно запрашивается прогноз на 5 дней, из которого выбирается одна запись на каждый день (на 12:00)
Интерфейс обновляется: иконка погоды, температура, статистика и фон карточки подстраиваются под полученные данные


Запуск локально

Клонируй репозиторий и открой index.html в браузере:

bashgit clone https://github.com/dex1tt/weather-app.git
cd weather-app

Если хочешь использовать свой собственный API-ключ — получи бесплатный ключ на openweathermap.org и подставь его в script.js:

javascriptconst API_KEY = "твой_ключ_здесь"
